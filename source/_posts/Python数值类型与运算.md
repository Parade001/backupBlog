##### 整数
常被称为整型或整数类型,不包含小数点的数字.

- 十进制： 0 、123、-45
- 二进制：0b1101、0B1100
- 八进制：0o17、0O777
- 十六进制: 0 x ff、0 x 123456789ABCDEF
  - 布尔类为整数的子类

##### 浮点数（float, 实数）
十进制表示
由整数部分、小数点与小数部分组成
